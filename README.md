# Gym Management System (Flask)

Features:
- Member management
- Active member count
- Search members
- PDF receipt generation
- Email receipt using Gmail SMTP
- Login system

Tech Stack:
- Flask
- SQLAlchemy
- SQLite
- Flask-Mail
- ReportLab

Setup:
1. Clone repo
2. Create virtual environment
3. Install requirements
4. Set MAIL_PASSWORD env variable
5. Run app
